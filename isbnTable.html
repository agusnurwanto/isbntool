How to use:

  1. Add new ISBNs number:
    • Upload CSV file
      • Setting your data excel like this file. Donwload here!.
      • Save as to .csv format.
      • Click Upload CSV button and select your CSV file. You can donwload example CSV file in here.
      • Apps will auto scrape the data.
    • Textarea
      • Insert ISBNs number to textarea field with coma spareated.
      • Click Submit button to run apps.
    • Manual
      • Click Add New number button. You will get a modal popup.
      • Fill all fields on the form and click save button.
  2. Edit data with click on edit button.
  3. Delete data with click on delete button.
